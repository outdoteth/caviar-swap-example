# Caviar swap examples

Example scripts for executing buys and sells.

## Getting started

Create a file `.env` at the root with the following contents:

```
ALCHEMY_API_KEY=<your api key>
PRIVATE_KEY=<your private key>
```

Run:

```
npm install
```

To buy:

```
node buy.js
```

To sell:

```
node sell.js
```
